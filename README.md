## What is Ellie?

**Ellie is the control plane for [Shre](https://github.com/Nirpat3/ellie) — a distributed neural AI operating system.**

Built on top of [Paperclip](https://github.com/paperclipai/paperclip) and powered by [OpenClaw](https://openclaw.dev), Ellie provides:

- **Agent Orchestration** — Manage a team of specialized AI agents with org charts, delegation, and reporting lines
- **Skills Intelligence** — Skill profiling, coverage matrix, and gap detection across your agent network
- **System Status** — Real-time health monitoring of all Shre services
- **Dark iOS Theme** — Pure black Shre dark theme with iOS design conventions
- **OpenClaw Gateway** — Pre-configured to connect to your local OpenClaw instance

## Quickstart

```bash
cd ~/Documents/Projects/shreai/ellie-ui-v2
pnpm install
pnpm dev
```

Open http://localhost:3100

## Based on Paperclip

Ellie is a customized fork of [Paperclip](https://github.com/paperclipai/paperclip) — an open-source orchestration platform for AI agent companies. All original Paperclip features (Dashboard, Agents, OrgChart, Goals, Issues, Costs, Activity, Approvals, Projects, Inbox, Kanban) are preserved.
